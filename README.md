## 🚀 What is This?

> **YouTube Full Clone** is a frontend project currently under active development — being built from scratch with pure **HTML**, **CSS**, and **JavaScript** to recreate the real YouTube experience. This is not just a copy-paste project — it's a carefully crafted **UI clone** focusing on clean code, real-world structure, and portfolio-quality presentation.

```bash

│
├── 📄 index.html          # Homepage

│
├── 📁 css/
```
