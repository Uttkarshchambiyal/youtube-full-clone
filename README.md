# 🎬 YouTube Full Clone

A fully responsive **YouTube clone** built with HTML, CSS, and JavaScript — replicating the core UI and features of YouTube's web interface.

## 🚀 Features

- 📺 YouTube-like homepage with video thumbnails grid
- 🔍 Search bar with responsive layout
- 🎞️ Video player page with autoplay and sidebar suggestions
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🌙 Dark/Light mode toggle
- 🔔 Header with notification bell, account, and upload icons
- 📂 Sidebar with navigation (Home, Shorts, Subscriptions, Library, etc.)
- 💬 Comments section UI
- 👍 Like, Dislike, Share, and Save buttons

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Styling, Flexbox, Grid |
| JavaScript (ES6+) | Interactivity & DOM manipulation |

## 📁 Project Structure
