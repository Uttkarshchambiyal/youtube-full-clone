## 🚀 Overview

This project is a **YouTube Full Clone** that recreates the familiar layout and user experience of the YouTube web app with a modern frontend approach. It focuses on building a realistic video platform interface with clean structure, responsive design, sidebar navigation, video cards, watch-page UI, search layout, and engaging visual styling.

Instead of just copying the look, this project is designed to feel like a real product showcase piece for a portfolio — visually sharp, responsive, and user-friendly.

## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/Uttkarshchambiyal/youtube-full-clone.git
```

2. Open the project folder
```bash
cd youtube-full-clone
```

3. Run it locally  
Just open `index.html` in your browser.
